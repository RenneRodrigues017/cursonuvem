# cursonuvem
Nesse curso aprendi sobre topicos especificos da nuvem, como escalibilidade, gereciamento, segurança, entre outros.
Topicos que sao essesciais para o aprendizado de como usar essa ferramenta para o nosso favor. Sem esses conheciomentos ao tentar utilizar uma nuvem como por exemplo na Microsoft Azure, voce pode e provavelmente tera custos que nao estavam no seu planejamento, e para qualquer empresa esse custo alt é totalmente inviavel, entao devesse estudar toda essa ferramente para a entrega mais efetiva possivel
